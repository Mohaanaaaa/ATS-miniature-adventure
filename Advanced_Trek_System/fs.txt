How to use this Test Kit:
Start Backend: Run python app.py in your backend folder.

Start Frontend: Run npm start in your frontend folder.

Register: On the React website, register a trekker with the ID 2026000100001.

Run Test: In a new terminal, run python test_simulation.py.

What to watch for (The "Advanced" Verification):
Breadcrumbs: Check if the CircleMarkers (the dots) appear exactly where the script sent the coordinates.

The "Alive" Logic: When Step 3 runs (HR=0), look at your React Sidebar. Does the status change to indicate the band is detached?

The SOS Logic: When Step 4 runs, does the map marker and the line turn RED instantly?
